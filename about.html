What is this project?

This project aims to find the shortest path from any Massachusetts USCF player to any other Massachusetts USCF player. The shortest path Between two players is considered to be the shortest number of winning games from player 1 to player 2, for example: player 1 beat 3, 3 beat 2 is a valid path from 1 to 2, and will be the shortest one if there has not been a game where 1 beat 2 themselves. My inspiration for the project was from seeing the Magnus Number and Chessbase projects with the same pathfinding idea as this one.

How did you implement the project?

I implemented the project in Python and MySQL. I first scraped all USCF rated tournaments for Massachusetts and the games within them into a MySQL database, after which I extracted all the games that had results and placed them into a Python graph using the networkx module. Then by using Djikstra's algorithm pathfinding algorithm inside the networkx module I was able to find the shortest path between two players, and output that path accordingly. The website was made using the Django framework.
